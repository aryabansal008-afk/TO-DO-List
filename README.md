# TO-DO-List
A simple and interactive To-Do List application built using HTML, CSS and JavaScript that helps users organize and manage their daily tasks efficiently. The application allows users to add, complete, delete and filter tasks through an intuitive interface with a sidebar navigation menu.


## 🚀 Features

- **Task Creation:** Users can add new tasks through an input field.
- **Task Completion:** Tasks can be marked as completed using a dedicated check button.
- **Task Deletion:** Individual tasks can be removed from the list.
- **Task Filtering:** Sidebar navigation allows users to view:
  - All tasks
  - Completed tasks
  - Pending tasks
- **Clear All Functionality:** Enables users to remove all tasks and reset the interface to its initial state.
- **Interactive Sidebar Menu:** A collapsible sidebar provides quick access to task filters.
- **Overlay Interaction:** When the sidebar is opened, the background interface becomes inactive through an overlay effect.
- **Responsive Interface:** The layout provides a clean and user friendly experience.
  

## 🛠️ Technology Stack

- **HTML5** – Provides the structural foundation of the application.
- **CSS3** – Handles styling, layout design and visual presentation.
- **JavaScript (Vanilla JS)** – Implements application logic, DOM manipulation and interactive behavior.
